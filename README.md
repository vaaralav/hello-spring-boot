# hello-spring-boot

## Getting started

### Requirements

- [spring-boot](https://projects.spring.io/spring-boot/)
- [Maven](https://maven.apache.org/)

### Commandz

#### `mvn spring-boot:run`
Start the application.

#### `mvn package`
Package `.jar`. To run the jar use `java -jar target/hello-spring-boot-0.1.0-SNAPSHOT.jar`
